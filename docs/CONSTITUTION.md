# 小黑书项目宪章

## UI 尺寸基准

- 课表界面以当前小黑书课表风格的紧凑尺寸为标准，不再放大组件。
- 新增或修改组件时，默认遵循这个密度：
  - 顶部日期约 24sp，周次文字约 15sp，顶部图标约 25sp。
  - 星期栏文字约 13sp。
  - 左侧节次栏宽约 42dp，每节高度约 56dp。
  - 节次数字约 15sp，时间约 10sp。
  - 课表块正文约 11sp，拥挤/并列课程约 9sp。
- 课程详情抽屉、弹窗、菜单等后续界面也要按同一紧凑密度设计，不能回到大字号、大间距、大图标。
- 如果需要增加信息量，优先调整层级、折叠、滚动和点击详情，不通过整体放大解决。

## 中石大导入基准

- 中石大课表导入必须走 HTTP/API 会话：统一认证登录、学期识别、课表详情请求都由 API 客户端完成。
- 导入页和账号测试不能展示内嵌网页、浏览器页面或自动填写网页表单。
- 点选学期后先展示课表详情摘要，再通过右下角确认按钮保存；本地已存在同学期课表时必须确认覆盖。
- 本地可以保存多个课表，右上角更多菜单负责设置当前课表并持久化。
